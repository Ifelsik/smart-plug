# Умная розетка

Исходный код умной розетки на базе микроконтроллера ESP32-S3-WROOM-1 (esp32-s3 N16R8) и модема SIMCom A7670E.

## Протокол взаимодействия

Устройство управляется по протоколу MQTT. Есть 2 топика очереди: 

 - `thesis/plug/command` - Топик управляющих команд. Устройство подписано на него.
 - `thesis/plug/state` - Топик состояний устройства, куда публикуются сообщения о статусах. Имеет QoS (Quality of Service) равный 1 - что гарантирует доставку at-least-once.

Устройство принимает команду по `thesis/plug/command`, обрабатывает её и отправляет результат в `thesis/plug/state`.

Обмен данными происходит в формате json.

### Управляющие команды

Управляющие команды имеют такой формат:

```json
{
    "cmd": <command>
}
```

где \<command> может иметь один из следующих вариантов:

 - `"status"` - получение статуса
 - `"relay_on"` - включение реле
 - `"relay_off"` - отключение реле
 - `"reboot"` - перезагрузка устройства

### Команды статуса

На каждую управляющую команду приходит асинхронный ответ по результатам обработки. Такие команды имеют следующий вид, например:

```json
{
    "cmd": "relay_on",
    "result": "ok",
    "desc": "relay switched on"
}
```

где:

 - `"cmd"` - исполненная команда
 - `"result"` - результат выполнения команды. Может быть `ok`, `error`, `partial`
 - `"desc"` - пояснение к результату (опционально, может быть пустым)
